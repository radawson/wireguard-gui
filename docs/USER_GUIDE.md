# User Guide

Use the following docs depending on your task:

- Installation: `docs/INSTALLATION.md`
- Upgrade existing deployment: `docs/UPGRADE.md`
- Runtime configuration reference: `docs/CONFIGURATION.md`
- Operations and recovery: `docs/OPERATIONS.md`

Primary UI flows:

1. Login as admin user.
2. Create or import networks under **Networks**.
3. Add and manage peers under **Peers**.
4. Use **Wizard** for guided basic setup.
